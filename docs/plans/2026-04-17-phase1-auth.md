# Phase 1 — Foundation & Auth Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Phase 0 목업을 Supabase 백엔드와 Google 로그인에 실제 연결해 "로그인 → 본인 프로필 + 색상 팔레트 조회 → /trips 접근" 흐름이 실 데이터로 동작하게 만든다. 데이터 CRUD(여행/일정/경비)는 Phase 2~5에서 다룬다.

**Architecture:**
- **Auth:** Google Identity Services(GIS, 클라이언트) → `supabase.auth.signInWithIdToken({ provider: 'google' })` → Supabase JWT 세션. `@supabase/ssr` 가 쿠키에 세션을 저장하고 Next.js middleware가 매 요청마다 refresh.
- **Data layer:** TanStack Query 가 캐싱·무효화 단일 진입점. Supabase 클라이언트는 browser/server/middleware 3종으로 분리(@supabase/ssr 가이드 준수). Zustand 는 UI 상태(시트 토글 등)만, 서버 데이터는 TanStack 만.
- **Profile color:** ADR-007 — `profiles.color` (NOT NULL, CHECK 6색, default 'orange'). `useProfile()` 훅이 본인+파트너 프로필을 캐싱하고 `chipClassForProfile(profile)` 가 토큰 매핑 반환. Phase 0 의 `currentUserId` 하드코딩 제거.

**Tech Stack:** Next.js 16.2.4 App Router · TypeScript strict · `@supabase/supabase-js@^2` · `@supabase/ssr@^0.5` · `@tanstack/react-query@^5` · `zustand@^5` · `zod@^3` · `vitest@^2` (단위) · `@playwright/test@^1.49` (E2E) · Supabase CLI (마이그레이션) · Google Identity Services (브라우저)

## Pre-flight (사용자 1회 작업, 코드 외부)

이 작업들은 사용자가 콘솔에서 직접 수행하고 결과 값(키·ID)을 `.env.local` 에 채운다. Task 1 에서 `.env.local` 골격을 생성한다.

### A. Supabase 프로젝트 생성
1. https://supabase.com/dashboard → "New project"
2. Name: `travel-manager-dev`, Region: `Northeast Asia (Seoul)`, DB password 안전한 곳에 보관
3. Settings → API 에서 다음 값 확보:
   - `NEXT_PUBLIC_SUPABASE_URL` (Project URL)
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY` (anon public)
   - `SUPABASE_SERVICE_ROLE_KEY` (서버 전용, 절대 클라이언트 노출 금지)

### B. Google OAuth Client ID 발급
1. https://console.cloud.google.com → 새 프로젝트 `travel-manager`
2. APIs & Services → Credentials → "Create Credentials" → OAuth Client ID → Web application
3. Authorized JavaScript origins: `http://localhost:3000`, `https://travel-manager-yeoullab.vercel.app`
4. Authorized redirect URIs: 비워둠 (GIS One Tap 은 redirect 없음)
5. Client ID 저장 → `NEXT_PUBLIC_GOOGLE_CLIENT_ID`

### C. Supabase 의 Google Provider 활성 (선택, signInWithIdToken 만 쓸 거면 사실상 미사용)
- `signInWithIdToken({ provider: 'google' })` 는 Supabase 의 GoTrue 가 ID token 의 `aud` claim 을 검증할 때 Authentication → Providers → Google 에 등록된 Client ID 와 매칭한다.
- Authentication → Providers → Google → Enabled ON, Client ID(B 의 값) 입력. Client Secret 는 비워도 동작 (ID token flow 라 secret 불필요).

### D. Supabase CLI 설치 및 링크
```bash
brew install supabase/tap/supabase   # macOS, 미설치 시
supabase --version                    # ≥ 1.200 권장
supabase login                        # 브라우저 로그인
```
프로젝트 링크는 Task 2에서 수행.

- [ ] **Step 1: 마이그레이션 파일 작성**

`supabase/migrations/0001_profiles.sql`:

```sql
-- 0001_profiles.sql
-- Phase 1: profiles 테이블 + auto-create trigger + RLS + 공개 view
-- 스펙 §5.profiles, §RLS profiles ("이메일은 본인만"), ADR-007 (color 6색 팔레트)

-- ── 테이블 ────────────────────────────────────────────────────────────
create table public.profiles (
  id uuid primary key references auth.users(id) on delete cascade,
  email text not null,
  display_name text,
  avatar_url text,
  color text not null default 'orange'
    check (color in ('orange','blue','gold','violet','green','rose')),
  created_at timestamptz not null default now()
);

comment on column public.profiles.color is
  'ADR-007 6색 팔레트. DESIGN.md 토큰과 1:1 매핑.';
```

```ts
import { NextRequest, NextResponse } from "next/server";
import { updateSession } from "@/lib/supabase/middleware-client";

const PROTECTED_PREFIXES = ["/trips", "/settings", "/invite"];

export async function middleware(request: NextRequest) {
  const { response, user } = await updateSession(request);

  const { pathname } = request.nextUrl;
  const isProtected = PROTECTED_PREFIXES.some(
    (prefix) => pathname === prefix || pathname.startsWith(`${prefix}/`),
  );

  if (isProtected && !user) {
    const url = request.nextUrl.clone();
    url.pathname = "/login";
    url.searchParams.set("redirect", pathname);
    return NextResponse.redirect(url);
  }

  return response;
}
```

```ts
function bytesToBase64Url(bytes: Uint8Array): string {
  let binary = "";
  for (const b of bytes) binary += String.fromCharCode(b);
  return btoa(binary).replace(/\+/g, "-").replace(/\//g, "_").replace(/=+$/, "");
}

export function generateNonce(): string {
  const bytes = new Uint8Array(32);
  crypto.getRandomValues(bytes);
  return bytesToBase64Url(bytes);
}

export async function sha256Base64Url(input: string): Promise<string> {
  const data = new TextEncoder().encode(input);
  const hash = await crypto.subtle.digest("SHA-256", data);
  return bytesToBase64Url(new Uint8Array(hash));
}
```

```ts
const GIS_SRC = "https://accounts.google.com/gsi/client";
let scriptPromise: Promise<void> | null = null;

export function loadGisScript(): Promise<void> {
  if (typeof window === "undefined") {
    return Promise.reject(new Error("GIS는 브라우저에서만 동작합니다"));
  }
  if (window.google?.accounts?.id) return Promise.resolve();
  if (scriptPromise) return scriptPromise;

  scriptPromise = new Promise<void>((resolve, reject) => {
    const existing = document.querySelector<HTMLScriptElement>(
      `script[src="${GIS_SRC}"]`,
    );
    if (existing) {
      existing.addEventListener("load", () => resolve());
      existing.addEventListener("error", () => reject(new Error("GIS 로드 실패")));
      return;
    }
    const script = document.createElement("script");
    script.src = GIS_SRC;
    script.async = true;
    script.defer = true;
    script.onload = () => resolve();
    script.onerror = () => reject(new Error("GIS 로드 실패"));
    document.head.appendChild(script);
  });
  return scriptPromise;
}

export type GoogleSignInResult = {
  idToken: string;
  rawNonce: string;
};

export async function requestGoogleIdToken(
  parent: HTMLElement,
): Promise<GoogleSignInResult> {
  await loadGisScript();
  if (!window.google) throw new Error("GIS 미로드");

  const rawNonce = generateNonce();
  const hashedNonce = await sha256Base64Url(rawNonce);

  return new Promise<GoogleSignInResult>((resolve, reject) => {
    try {
      window.google!.accounts.id.initialize({
        client_id: env.NEXT_PUBLIC_GOOGLE_CLIENT_ID,
        nonce: hashedNonce,
        use_fedcm_for_prompt: true,
        callback: (response) => {
          if (!response.credential) {
            reject(new Error("Google 응답에 credential 이 없습니다"));
            return;
          }
          resolve({ idToken: response.credential, rawNonce });
        },
      });
      parent.innerHTML = "";
      window.google!.accounts.id.renderButton(parent, {
        type: "standard",
        theme: "outline",
        size: "large",
        text: "continue_with",
        shape: "rectangular",
        width: parent.clientWidth || 320,
      });
    } catch (error) {
      reject(error instanceof Error ? error : new Error("GIS 초기화 실패"));
    }
  });
}
```

```ts
"use client";

import { useQuery } from "@tanstack/react-query";
import { getBrowserClient } from "@/lib/supabase/browser-client";
import { queryKeys } from "@/lib/query/keys";
import type { Database } from "@/types/database";

// 본인용: profiles 테이블 (이메일 컬럼은 GRANT 에서 제외되어 안 옴)
export type MyProfile = Database["public"]["Tables"]["profiles"]["Row"];
// 타인용: profiles_public view (display_name, avatar_url, color, id, created_at)
export type PublicProfile = Database["public"]["Views"]["profiles_public"]["Row"];

export function useMyProfile() {
  const supabase = getBrowserClient();
  return useQuery({
    queryKey: queryKeys.profile.me,
    queryFn: async (): Promise<MyProfile | null> => {
      const { data: { user } } = await supabase.auth.getUser();
      if (!user) return null;
      const { data, error } = await supabase
        .from("profiles")
        .select("id, display_name, avatar_url, color, created_at")
        .eq("id", user.id)
        .single();
      if (error) throw error;
      return data as MyProfile;
    },
  });
}

// useProfileById — 단발 조회 (settings, header 등에서 1명만 볼 때).
// 행 안에서 호출 금지 — N+1 위험. 행 안에서는 useTripMembers 의 lookup 함수 사용.
export function useProfileById(id: string | null | undefined) {
  const supabase = getBrowserClient();
  return useQuery({
    queryKey: queryKeys.profile.byId(id ?? "__none__"),
    enabled: !!id,
    queryFn: async (): Promise<PublicProfile | null> => {
      if (!id) return null;
      const { data, error } = await supabase
        .from("profiles_public")
        .select("*")
        .eq("id", id)
        .maybeSingle();
      if (error) throw error;
      return data;
    },
  });
}
```

```ts
"use client";

import { useMemo } from "react";
import { useQuery } from "@tanstack/react-query";
import { getBrowserClient } from "@/lib/supabase/browser-client";
import { queryKeys } from "@/lib/query/keys";
import type { PublicProfile } from "@/lib/profile/use-profile";

/**
 * Trip 에 등장 가능한 모든 프로필을 한 번에 fetch.
 * Phase 1: trip → group_members 가 아직 mock. profiles_public 전체를 가져오는 임시 구현.
 *          (현재 본인 + 가입한 다른 유저 몇 명 정도라 비용 무시 가능)
 * Phase 6: trip.group_id → group_members → user_id IN (...) 로 좁힘.
 */
export function useTripMembers(tripId: string | null | undefined) {
  const supabase = getBrowserClient();
  const query = useQuery({
    queryKey: queryKeys.tripMembers.byTripId(tripId ?? "__none__"),
    enabled: !!tripId,
    staleTime: 5 * 60 * 1000, // members 는 자주 안 바뀜
    queryFn: async (): Promise<PublicProfile[]> => {
      const { data, error } = await supabase
        .from("profiles_public")
        .select("*");
      if (error) throw error;
      return data ?? [];
    },
  });

  const lookup = useMemo(() => {
    const map = new Map<string, PublicProfile>();
    for (const p of query.data ?? []) map.set(p.id, p);
    return (id: string | null | undefined): PublicProfile | null => {
      if (!id) return null;
      return map.get(id) ?? null;
    };
  }, [query.data]);

  return { ...query, lookup };
}
```

```tsx
  userId: string;

export function ColorPalette({ currentColor, userId }: Props) {

    setSelected(color);
    mutation.mutate(color);

      <legend className="text-ink-700 text-[13px] font-medium">프로필 색상</legend>

              key={color}

              )}
            />
          );
        })}
      </div>
```

```ts
"use client";

import { getBrowserClient } from "@/lib/supabase/browser-client";
import type { RealtimePostgresChangesPayload } from "@supabase/supabase-js";

type SubscribeOptions<Row> = {
  channel: string;
  table: string;
  filter?: string;
  onChange: (payload: RealtimePostgresChangesPayload<Row>) => void;
};

export function subscribeToTable<Row extends Record<string, unknown>>({
  channel,
  table,
  filter,
  onChange,
}: SubscribeOptions<Row>) {
  const supabase = getBrowserClient();
  const ch = supabase
    .channel(channel)
    .on(
      "postgres_changes",
      { event: "*", schema: "public", table, filter },
      (payload) => onChange(payload as RealtimePostgresChangesPayload<Row>),
    )
    .subscribe();

  return () => {
    void supabase.removeChannel(ch);
  };
}
```
